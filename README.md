# Project Hello World

Simple Node.js web server that renders a `Hello World` page.

## Prerequisites

- Node.js 18+ (comes with `npm`)

## Setup

No dependencies are required. Just clone/copy this folder.

## Run

```bash
npm start
```

The server listens on `http://localhost:3000` (or the port defined in `PORT`).

## Deploying

Any Node-friendly host works:

1. Upload the project files.
2. Set the start command to `npm start`.
3. Expose port `3000` or configure `PORT` via environment variables.

You should see `Hello World` centered on the page.
